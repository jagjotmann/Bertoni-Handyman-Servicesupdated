## Run Locally

Clone the project

```bash
  git clone https://github.com/seanthecod3r/Bertoni-Handyman-Services.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  cd Backend && npm install
```

```bash
  cd Frontend/bertoni-handyman-services && npm install
```

Start the server

```bash
  npm run start
```
